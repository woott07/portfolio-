# Md Sowel Rana | Portfolio

> "I Dlulu into Solulu."

Welcome to my personal portfolio repository! This project showcases my skills, projects, and the automation tools I build. Rather than doing side quests, I patch the main mission.

## 🚀 Features

- **Modern UI/UX**: Glassmorphism design, custom cursor effects, and smooth reveal animations.
- **Interactive Canvas Background**: Features a unique, lightweight particle and shooting star animation.
- **Responsive Design**: Fully optimized for both desktop and mobile viewing.
- **Project Showcase**: Detailed modals for viewing project descriptions, tech stacks, and live demos.
- **Dynamic Skill Bars**: Animated progress bars to highlight technical proficiencies.

## 🛠️ Tech Stack

- **HTML5 & CSS3**: Vanilla setup relying on CSS variables, Flexbox, CSS Grid, and modern animations. 
- **Vanilla JavaScript**: DOM manipulation, Intersection Observers for scroll animations, Canvas API for the background, and customized cursor logic.
- **Feather Icons**: Minimalist, open-source icons.

## 💼 Featured Projects

1. **Omni Bot** - Advanced Discord moderation and utility bot featuring custom commands and deep logging. *(Python, Discord.py, Automation, Spotify, FFmpeg, APIs)*
2. **Data Pipeline (WebScraping)** - Automated data extraction from websites with secure storage. *(Python, APIs, Automation, Data)*
3. **Task Automator** - Background utility for automated file management and reporting. *(Python, Scripting, Numpy, Pandas)*
4. **Universal Downloader** - A tool to download videos, audio, galleries, or scraped webpage images from 1000+ sites directly to your browser without local storage overhead. *(Python, Flask, yt-dlp, gallery-dl, BeautifulSoup4)*

## 📥 Local Setup

Since this is a static website built with pure HTML, CSS, and JS, no complex build process is required!

1. Clone the repository to your local machine:
   ```bash
   git clone https://github.com/woott07/portfolio.git
   ```
2. Navigate into the folder and simply open the `index.html` file in your favorite modern browser.
